## Description

Shopping cart implementation with nest.js. 

## Before first run

1. Copy .env.example to .env and fill it with your data

## Running with docker

```bash
$ docker-compose up --build -V -d
```

## Running without docker

```bash
#install
$ npm install

# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
