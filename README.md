## Description

Shopping cart implementation with nest.js. 

## Before first run

1. Run `npm install`
2. Copy `.env.example` to `.env` and fill it with your data
3. Run tests `npm run test`

## Running with docker

```bash
$ docker-compose up --build -V -d
```

## Running without docker

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
